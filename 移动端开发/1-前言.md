# 前言

### 移动端发展历史

页面演进

* 设备

  诺基亚 -> iPhone -> iPhone X

* 系统

  Java EE -> Symbian -> iOS, Android, Windows

* 网络

  2G -> 2.5G -> 3G,4G

* 语言

  WML (Wireless Markup Language) -> XHTML -> XHTML MP (Mobile Profile)-> HTML5

### 移动端和 PC 端的差异

1.  浏览器差异

    PC : IE , Safari, Chrome, Firefox

    移动端: Safari, Chrome, webview

1.  页面结构

    PC: 横向式 (屏幕宽)

    移动端: 纵向式

1.  操作方式

    PC: 鼠标

    移动端: 触摸
