* 网络优化

  如果页面白屏 3 秒; 那么我们 50% 的用户可能已经离开这个页面了;

  1.  `0.5 * 0.5 * ...`

  链路越长, 转化越差; 影响销售

  会影响到 SEO; 影响曝露, 影响流量;

1.  定位性能问题

首屏时间 = DNS 时间 (DNS Lookup) + 建立连接时间 (Inital Connection) + 后端响应时间(TTFB 首字节返回) + 网络传输时间 (Content Download) + 首屏页面渲染时间

DNS 与 CDN 相关, 如果是单机部署, 查询时间会高一些;

TTFB 与后端响应相关

Content Download 与网络环境, 文件大小有关
