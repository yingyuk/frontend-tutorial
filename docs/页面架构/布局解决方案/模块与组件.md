### 如何实现模块化

模块化一般在 一个容器内, 是一个整体;
如导航模块

```html
<style>
.nav{
  ...
}
</style>
<div class="nav">
  ...
</div>
```

当遇到 外观相似, 语义相同的模块, 可以基于一个基本模块进行拓展;
一个相似的导航模块:

```html
<style>
.nav{
  ...
}
.nav .nav-other{
  ...
}
</style>
<div class="nav nav-other">
  ...
</div>
```
